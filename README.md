## Featured Work

### EcoFunding — Funding Discovery & Evaluation Platform

Case study of a platform developed during my internship at Microsoft for the Instituto de Conservación de Ballenas (ICB), an NGO dedicated to whale conservation in Patagonia and the South Atlantic.

The system automated the discovery and evaluation of international funding opportunities through a decoupled pipeline architecture.

Key aspects of the project included:

- Web scraping and automated discovery pipelines
- Semantic evaluation using sentence-transformers embeddings
- Hybrid scoring system combining heuristics and semantic similarity
- Classification of relevant opportunities vs news/content noise
- FastAPI backend with PostgreSQL and JWT authentication
- React frontend for operational management
- Docker-based deployment on Azure Container Apps
- CI/CD workflows using GitHub Actions

The platform reduced manual research time from approximately 15 hours to 2 hours per week.

> Source code remains private due to organizational ownership and internal infrastructure configuration.
